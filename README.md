# Stats

### vpip (voluntarily put money in pot)

- any call pf
- any raise pf
- bb check does not count as vpip

### 3b (3 betting)

- 3 betting open raise pf
- cases:
  - limp -> 3b (update player)
  - straight 3b (add player)

### PFR (preflop raise)

- first raise preflop
- player will always be new when making first pfr
